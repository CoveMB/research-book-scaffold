# Bibliography

Keep citation source files here.

Primary file:

```text
references.bib
```

Zotero or this BibTeX file is the citation source of truth. Do not add citations from memory.

## Basic workflow

1. Add or verify sources in Zotero.
2. Export or sync BibTeX into `references.bib`.
3. Use stable citekeys in notes and manuscript files.
4. Run citation checks before sharing drafts.
