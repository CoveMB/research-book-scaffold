# Claim ledger

Claims, evidence status, confidence, and follow-up research tasks.

Use this folder for proposed claims before they are promoted into manuscript
prose.

Include claim text, claim type, evidence status, confidence, supporting sources
with locators, counter-sources, safer wording, and the next research task.

Do not mark a claim as ready because it sounds plausible. Claims need a visible
path back to source notes, citekeys, locators, and any relevant counterevidence.
