# Writing bridge

Use `40-writing-bridge/` for notes that prepare research for manuscript drafting.

This layer does not hold final prose. It translates checked evidence, claims,
argument maps, and open gaps into a chapter or section plan.

Subfolders:

- `chapter-briefs/` for chapter and section purpose, argument sequence, evidence
  placement, likely sources, counterarguments, definitions, and gaps.

Use this layer when a set of claims is ready to be organized for writing. Keep
gaps visible so the manuscript does not absorb unsupported claims.

Draft in `manuscript/` only after the relevant source notes, claim notes, and
briefs support the move.
