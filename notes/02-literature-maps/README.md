# Literature maps

Maps of debates, schools of thought, gaps, and source relationships.

Use this folder when the unit of analysis is a field, debate, school of thought,
or source cluster rather than one source.

Include the sources being compared, the relationship being mapped, points of
agreement and disagreement, corpus limits, gaps, and follow-up searches. Link
back to source notes, extraction tables, source matrices, and relevant claim IDs.

Do not use a literature map to invent a consensus or smooth over disagreement.
If a relationship is inferred rather than source-supported, label it as
provisional.
