# Synthesis memos

Use `synthesis-memos/` for cautious cross-source interpretation.

Create a synthesis memo when several source notes, case notes, extraction tables,
source matrices, or literature maps are ready to be read together. Use
`templates/synthesis-memo-template.md` when it fits.

Include:

- The question being synthesized.
- Strongest evidence and strongest counterevidence.
- Corpus limits and uncertainty.
- Manuscript implications, if any.
- Links back to the notes and research records that support the synthesis.
- Next steps.

A synthesis memo is still provisional. Convert supported conclusions into claim
notes before drafting them as manuscript claims.
