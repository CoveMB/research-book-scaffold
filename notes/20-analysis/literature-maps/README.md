# Literature maps

Use `literature-maps/` when the unit of analysis is a field, debate, school of
thought, method cluster, or source group.

Create a literature map after several sources have been selected, read, or
screened closely enough to compare. It can draw on source notes, extraction
tables, source matrices, and search logs.

Include:

- The sources or source groups being compared.
- The relationship being mapped.
- Points of agreement and disagreement.
- Corpus limits, missing voices, and follow-up searches.
- Links to source notes, extraction tables, source matrices, and claim IDs.

Do not use a literature map to invent consensus or hide disagreement. If a
relationship is inferred rather than source-supported, label it as provisional.
