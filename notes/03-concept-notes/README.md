# Concept notes

Working definitions, related sources, and risks of misuse.

Use this folder for terms, concepts, distinctions, and boundaries that need to
stay stable across notes, claims, briefs, and chapters.

Include the working definition, competing definitions, related sources, what the
concept cannot carry, and risks of overextension or misuse.

Do not let a concept note become a claim ledger entry. When a definition supports
a manuscript claim, link the claim to source notes and citekeys with locators.
