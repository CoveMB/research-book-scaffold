# Evidence

Use `10-evidence/` for notes whose main job is to preserve the source trail.

This layer comes after capture and before cross-source analysis. Keep it close to
the material: one source, one case, one archive item, one interview, one site, or
one bounded event. The point is to make later claims checkable.

Subfolders:

- `source-notes/` for notes tied to specific sources and citekeys.
- `case-studies/` for bounded cases, examples, organizations, sites, or events.

Use this layer when the material still needs careful handling as evidence. Move
patterns, debates, definitions, and provisional interpretation into
`20-analysis/` once several evidence notes need to be compared.

Do not use this folder for unsupported claims or polished prose. If a quotation,
page number, URL, archive link, or citekey is missing, leave the gap visible.
