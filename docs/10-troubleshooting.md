# Troubleshooting

Common issues:

- Missing CLI tools: run the setup check once implemented.
- Missing bibliography: create or export `bibliography/references.bib`.
- Broken citekeys: compare draft citekeys with bibliography keys.
- Broken wiki links: run the link checker once implemented.
- Placeholders remain: run the placeholder checker once implemented.
- Export failed: check Quarto and Pandoc installation.
- Agent edited too much: review Git diff and restore only intended files.
